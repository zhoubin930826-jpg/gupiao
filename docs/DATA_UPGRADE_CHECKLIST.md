# 数据升级清单

这份清单专门回答一个问题：

**为了让 A 股推荐结果更稳、更周到，下一步最该补哪些数据。**

先说当前已经有的主支撑：

- A 股快照
- A 股日线历史
- 财务快照
- 第一版事件层
- 市场状态 / 风格环境
- 推荐诊断、涨跌归因、推荐复盘

它已经足够做一个“个人研究候选池系统”，但离“更像完整决策辅助系统”还差几层关键数据。

## 总优先级

如果只按价值排序，最推荐先补这四类：

1. 市场环境与风格数据
2. 资金流与筹码结构
3. 更完整的事件层
4. 更真实的交易约束与验证数据

## A股

### 当前已经有

- 快照
- 日线历史
- 财务摘要
- 公告 / 业绩预告事件层
- 推荐复盘

### 最值得补

#### P0

- 指数环境
  - 上证指数、深证成指、创业板指、沪深 300、中证 1000
- 市场宽度
  - 上涨家数、下跌家数、涨停、跌停、创新高、创新低
- 北向资金
  - 当日净流入、行业分布、个股净流入
- 龙虎榜
  - 上榜原因、买卖席位、净买额
- 主力资金
  - 个股净流入、行业净流入

#### P1

- 融资融券
- 业绩快报 / 正式财报节奏
- 公告分类细化
  - 回购、减持、订单、中标、诉讼、监管问询
- 一致预期与研报评级变化

### 业务价值

- 让系统更会判断“今天该不该做”
- 让推荐从“走势强”升级成“走势强且有资金和事件支撑”
- 让 A 股推荐更像真正的题材和情绪驱动分析

## 交易层还缺的数据

当前还建议继续补：

- 真实手续费 / 税费
- 滑点模型
- 交易限制
  - A 股涨跌停
  - A 股 T+1
- 更真实的回测成交规则

这部分不是为了“看起来更专业”，而是为了防止推荐复盘过于乐观。

## 推荐实现顺序

如果按“投入产出比”来排，我建议这样做：

1. A 股市场宽度 + 指数环境
2. A 股北向 / 龙虎榜 / 主力资金
3. 公告分类细化 + 业绩快报 / 正式财报节奏
4. 一致预期 + 研报评级变化
5. 真实交易约束与回测成交规则

## 当前这轮已经补上的第一步

这次已经先把：

- `市场状态 / 风格环境`

正式接进了系统看板。

它还不是新的外部源，但已经能明显改善推荐解释，因为系统终于能先回答：

- 今天是不是顺势环境
- 今天是主线抱团、轮动扩散，还是偏谨慎
- 高分票更适合进攻还是观察
